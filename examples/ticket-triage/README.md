# Ticket Triage Example

Support-ticket classifier example for EvalGate.

Files:
- [dataset.jsonl](./examples/ticket-triage/dataset.jsonl)
- [config.evalgate.json](./examples/ticket-triage/config.evalgate.json)
- [demo-pass.evalgate.json](./examples/ticket-triage/demo-pass.evalgate.json)
- [demo-fail.evalgate.json](./examples/ticket-triage/demo-fail.evalgate.json)

Run:

```bash
pnpm evalgate:sample
pnpm demo:pass
pnpm demo:fail
```
